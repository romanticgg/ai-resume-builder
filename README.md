# AI Resume Builder（应届生简历生成系统）

这是一个我自己做的简历生成小项目，主要是为了解决一个很现实的问题：
很多应届生其实有经历，但不会写简历。

这个系统可以把用户输入的原始经历，自动整理成一份结构清晰、表达更专业的简历。

---

## 项目做了什么

* 提供一个表单页面，让用户填写：

  * 基本信息
  * 求职意向
  * 教育背景
  * 项目/活动/竞赛经历
  * 实习经历
  * 技能和自我评价

* 点击生成后，系统会：

  * 自动整理结构（类似真实简历格式）
  * 对用户输入的内容做简单优化
  * 根据不同类型经历（如技术、文案、行政、活动等）调整表达方式

---

## 一个简单例子

用户输入：

> 负责活动签到和物资分发

系统输出：

> 参与活动现场组织与执行工作，负责签到管理与物资分发，保障活动流程顺利进行，体现了较强的执行能力和组织协调能力。

---

## 技术实现

后端用的是 Flask，负责处理表单数据和生成简历内容。
前端是纯 HTML + CSS + JavaScript，没有用框架，主要是为了简单直接。

项目里的“优化”不是用大模型，而是我自己写的一些规则，比如：

* 通过关键词判断经历类型（比如技术 / 文案 / 行政 / 活动等）
* 用不同模板去重写描述
* 做一些简单的语句补全

---

## 目前实现的功能

* 表单输入 + 简历生成
* 出生年月自动计算年龄
* 行业 / 地区 / 学历等下拉选择
* 支持多类型经历（不只限计算机专业）

---

## 可以改进的地方

这个项目目前还比较基础，后面可以继续优化：

* 支持多个项目/实习动态添加
* 导出 PDF 简历
* 页面做成更接近真实简历排版
* 引入大模型做更自然的语言优化

